# ポーカーレンジ表管理ツール

## Commands
- `open index.html` でブラウザ起動（サーバー不要）

## Architecture
- 純粋な HTML + CSS + JavaScript（フレームワークなし）
- 単一ファイル or 最小構成
- モバイルファーストで設計

## レンジ表の仕様
- ストリート：プリフロップ / フロップ / ターン / リバー
- ハンド：AKs, QQ などの表記を使用
- アクション割合：フォールド / コール / レイズ の%で管理
- フロップ以降はボードカード（3〜5枚）も登録が必要

## ハンド履歴の仕様
- 日時・ハンド・ストリート・実際のアクションを記録
- ローカルストレージに保存（サーバーなし）

## UI の方針
- スマートフォンで見やすいシンプルなデザイン
- タップ操作で完結
- 余計なライブラリは使わない

## Conventions（慣例）
- データはすべて localStorage で管理
- 外部APIや通信なし
- コメントは日本語でOK

## 実装済み機能

### レンジ表ページ
- **ストリート切替**：プリフロップ / フロップ / ターン / リバー
  - ストリートを切り替えてもボードカードは保持される
- **ポジション選択**：UTG / MP / CO / BTN / SB
  - 選択するとプリフロッププリセットが自動適用される
  - `?` アイコンでポジション説明ダイアログを表示
- **自分のハンド入力**：ホールカード2枚をタップで入力
  - 入力するとグリッド内の対応セルが黄色でハイライト
  - ハンドキー（AKs, QQ など）をバッジ表示
- **ボードカード入力**：フロップ以降で表示。ストリートに応じて3〜5枚
- **インライン記録**：ホールカード入力後にフォールド/コール/レイズボタンが出現し、タップで即時記録 → トースト通知
- **ボードテクスチャバッジ**：ボード完成時に自動分析して表示
  - スーツ：モノトーン / ツートーン / レインボー
  - 連続性：コネクテッド / セミコネクテッド / アンコネクテッド
  - ペアボード / Aハイ / Kハイ など
  - `?` アイコンで各バッジの説明ダイアログを表示
- **表示モード切替**（グリッド上部トグル）
  - **レンジ**：登録したアクション割合をカラーバーで表示（デフォルト）
  - **カテゴリー**：ボードに対する各ハンドの強さを色分け表示（フロップ以降）
    - セット / ツーペア / オーバーペア / トップペア / ミドルペア / ボトムペア / アンダーペア / FD / OESD / ガットショット / オーバーカード / エア
  - **統計**：履歴から同ストリート・同ボードのアクション割合を集計表示（件数付き）
  - `?` アイコンで各モードの説明ダイアログを表示

### プリフロップ プリセット（設定ページ）
- **戦略**：TAG / GTO を選択（デフォルト GTO）
- 169ハンド分のレイズ/コール/フォールド割合を自動登録
- GTO は混合戦略（100% / 67% / 50% / 33% のランダマイズ）

### ハンド履歴ページ
- ストリート・ホールカード・ボード・アクション・ポジション・日時を記録
- カードスロットのタップ入力（ランク → スーツの2ステップ）
- 個別削除対応

### カードピッカー（共通）
- 全コンテキスト共通のボトムシート UI
- ランク選択 → スーツ選択の2ステップ
- 使用済みカードを自動でグレーアウト

### データ管理（設定ページ）
- JSON エクスポート / インポート
- 全データ削除

## データ構造

### localStorage キー
- `pokerrng_range`：レンジデータ（ストリートボードキー → ハンドキー → {raise, call, fold}）
- `pokerrng_history`：履歴データ（配列）

### ストリートボードキー形式
- プリフロップ：`"preflop"`
- フロップ以降：`"flop:A♥-7♠-3♦"` （ランク+スーツ を `-` 区切り）

### ハンドキー形式
- ペア：`"AA"` `"KK"`
- スーテッド：`"AKs"` `"T9s"`
- オフスーツ：`"AKo"` `"T9o"`
